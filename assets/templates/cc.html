Cyclomatic Complexity Analysis


Cyclomatic Complexity describes the code complexity. The higher the number, the more confusing, unmanageable and incomprehensible the source code is.

Cyclomatic Complexity - Function Length Combined


The following picture illustrates a combined metric for the last tagged source code snapshot. The X axis visualizes the Cyclomatic Complexity values, while the Y axis represents the functionl length (NLOC). If there are many flaws in the right corner, this indicates spaghetti code with many branches. Most entries should be on the left side: the functions are well encapsulated and separated. If many values have high NLOC values, then there are many long functions. Very bad are plots with lots of values in the upper left corner: long functions with many branches.
